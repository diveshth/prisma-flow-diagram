# PRISMA Flow Diagram Generator

A Streamlit web application to generate PRISMA (Preferred Reporting Items for Systematic Reviews and Meta-Analyses) flow diagrams.

## Features

- Automatic calculation of intermediate values
- Interactive diagram generation
- Download diagrams as PNG files
- User-friendly interface

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/prisma-flow-diagram.git
cd prisma-flow-diagram
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## Usage

1. Choose between Auto Calculator or Manual Input
2. Enter your numbers
3. Click 'Generate PRISMA Flow Diagram'
4. Download the generated diagram
